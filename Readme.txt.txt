Pour installer l'application, il suffit juste d'installer le setup.

OpsieProtect\Main\bin\Debug\OpsieProtect

Les setups de ces différents modules sont présent dans le répertoire installé.

xxxxxxxxxxxxxxxxxxNOTESxxxxxxxxxxxxxxxxxxxxx
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Cette application est à utiliser sur des réseaux éthernet. En réseau wifi, il peut y avoir des disfonctionnements.
En particulier, s'il y a des machines virtuelles sur le réseau.
Le problème venant de la résolution d'adresse MAC qui ne se fait pas en Wifi sur les machines virtuelles.
De ce fait, la machine virtuelle dialoguera avec l'adresse MAC de la machine hôte.
